# Contributing

## Scope

Contributions should improve catalog data quality, documentation, and user experience.

## Adding A Listing

- Use the schema in `schema/integration.schema.json`.
- Include accurate support expectations and maintainer handles.
- Provide valid `repo_url`, `docs_url`, and `issue_url` links.

## Pull Requests

- Keep changes focused.
- Explain why each listing change is needed.
- Pass validation and link checks in CI.
